# Student Art NFT Gallery

NFT library of student artwork and creative expressions for school fundraising.

## Overview

A blockchain-based platform where students can mint their artwork as NFTs, sell them to support school fundraising efforts, and build their creative portfolios. Each sale automatically distributes proceeds between the artist and designated school programs.

## Features

- Student artist profile creation
- NFT minting for artwork
- Configurable school fundraising percentage
- Transparent sales tracking
- Artist earnings management
- Secondary market support

## Smart Contract Functions

### Public Functions
- `create-artist-profile` - Create a student artist profile
- `mint-artwork` - Mint artwork as an NFT with metadata
- `purchase-artwork` - Buy an NFT and support school fundraising
- `set-artwork-for-sale` - List or delist artwork for sale

### Read-only Functions
- `get-artwork` - Retrieve artwork details
- `get-artwork-owner` - Check current NFT owner
- `get-artist-profile` - View artist statistics
- `get-school-fundraising` - See total funds raised for school
- `get-total-artworks` - Get total NFTs minted

## Getting Started

1. Create your artist profile
2. Mint your artwork as an NFT with URI and pricing
3. Set a percentage for school fundraising
4. List your artwork for sale
5. Track your earnings and school contributions
